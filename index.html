Array Methods 1

  • Loads of array methods used for perusing and manipulating data
  • Array.Filter() - Filters the objects and finds when the result meets a specified parameter. I.e if (object.age >= 10) then show me those objects
  • Array.Map() - Maps through the list and outputs only the values that you're interested in seeing. Can use this to join different values, like first name and last name to make full name
  • Array.sort() - sorts the objects into an order using (a, b) - for example if (a.year is less than b.year) then return -1 else return 1. The numbers are assigned to a, b and this sorts the list.
  • Array.reduce() - The reducer walks through the array element-by-element, at each step adding the requested value to the result from the previous step — until there are no more elements to add.
  • Filtered through a list of elements (can choose any number of elements) in the dom and used filter and map to find those containing 'de' or whatever string you want!

Javascript reference vs copy

  • If you take an array, create a var referencing that array and then update the var (i.e. var[1] = "hello") that will actually update the main array too!
  • If you make a copy of that array using spread (const newArray = [...oldArray]) you can edit the new array but retain original
  • Same as above applies to objects. You can use Object.assign({}, objectName, { newKey = value } - this creates new empty object, then takes name of old object and fills new object with those key/values and then allows you to pass in new key/values. This will preserve original object
  • Use JSON.parse(JSON.stringify(yourObjectVar)); to turn an object into a string and back into an object, thus removing any mapping and making a clean copy.
