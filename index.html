Javascript reference vs copy

>

  • If you take an array, create a var referencing that array and then update the var (i.e. var[1] = "hello") that will actually update the main array too!
  • If you make a copy of that array using spread (const newArray = [...oldArray]) you can edit the new array but retain original
  • Same as above applies to objects. You can use Object.assign({}, objectName, { newKey = value } - this creates new empty object, then takes name of old object and fills new object with those key/values and then allows you to pass in new key/values. This will preserve original object
  • Use JSON.parse(JSON.stringify(yourObjectVar)); to turn an object into a string and back into an object, thus removing any mapping and making a clean copy.
